## 1. 제품 개요

### 한 줄 요약
YouTube 영상을 AI로 분석하여 팀이 함께 활용할 수 있는 구조화된 콘텐츠를 생성하는 B2B 도구.

### 해결하는 문제
콘텐츠 크리에이터와 마케터는 유튜브 영상 리서치에 많은 시간을 소비합니다. 1시간 영상의 핵심을 파악하려면 영상을 직접 봐야 하고, 이를 팀과 공유하려면 별도 정리가 필요합니다. ScriptFlow는 이 과정을 자동화하여 영상 → 구조화된 문서 → 팀 공유까지 한번에 해결합니다.

### 핵심 가치
- **자동 구조화**: 단순 요약이 아닌, 타임스탬프 기반 섹션 분리 + 핵심 포인트 + 키워드 추출
- **팀 공유**: Lilys AI에 없는 팀 워크스페이스. 분석 결과를 폴더별로 정리하고 댓글로 협업
- **커스텀 변환**: 회사의 브랜드 톤에 맞는 블로그/보고서/발표 대본으로 즉시 변환
- **로그인 없이 사용**: 접근 코드만으로 팀 워크스페이스 참여 가능

## 2. 사용자

### 주요 사용자
- **누구**: 콘텐츠 크리에이터, 마케터, 콘텐츠 팀 리드
- **상황**: 경쟁사/트렌드 영상을 리서치하고 이를 기반으로 콘텐츠를 기획할 때
- **목표**: 영상의 핵심을 빠르게 파악하고, 팀원에게 공유하며, 재가공 가능한 형태로 변환

### 사용자 시나리오

**시나리오 1: 개인 분석**
1. 마케터가 경쟁 채널의 영상 URL을 ScriptFlow에 입력
2. AI가 자막을 추출하고 구조화된 요약 생성
3. 타임스탬프별 섹션을 확인하며 핵심 구간 파악
4. 블로그 포스트 형태로 변환하여 콘텐츠 초안 완성

**시나리오 2: 팀 협업**
1. 팀 리드가 워크스페이스를 만들고 접근 코드를 팀원에게 공유
2. 팀원들이 각자 영상을 분석하여 "Q2 콘텐츠 리서치" 프로젝트에 저장
3. 팀원들이 분석 결과에 댓글로 의견 교환
4. 커스텀 템플릿으로 회사 블로그 스타일에 맞게 변환

## 4. 사용자 흐름 (User Flow)

### 핵심 흐름
```
워크스페이스 접속 -> 프로젝트 선택/생성 -> URL 입력 -> AI 분석 -> 결과 확인 -> 변환/공유
```

### 상세 흐름
1. **워크스페이스 접속**: 접근 코드 입력 또는 신규 워크스페이스 생성
2. **프로젝트 관리**: 폴더 생성, 분석 결과 정리
3. **영상 분석**: YouTube URL 입력 → 자막 추출 → AI 요약
4. **결과 확인**: 타임스탬프별 섹션, 핵심 포인트, 키워드 확인
5. **변환/내보내기**: 커스텀 템플릿 적용 → 블로그/보고서로 변환 → PDF 다운로드
6. **팀 협업**: 댓글 작성, 태그 추가, 팀원 피드백

## 5. 성공 기준

- [ ] 워크스페이스를 만들고 접근 코드로 3명 이상이 동시 접속할 수 있다
- [ ] 분석 결과가 DB에 저장되어 재방문 시 유지된다
- [ ] 커스텀 템플릿을 만들어 저장하고 다른 영상에 적용할 수 있다
- [ ] 댓글로 팀원 간 피드백을 주고받을 수 있다
- [ ] Vercel에 배포되어 외부에서 URL로 접속 가능하다

## 6. 안 만드는 것 (Out of Scope)

> 이 목록에 있는 건 Phase 1에서 만들지 않습니다.

- 사용자 로그인/회원가입 -- 이유: 접근 코드 방식으로 충분. MVP에서 인증 복잡도 제거
- 파일 업로드 분석 -- 이유: Phase 2에서 추가. 음성→텍스트 변환 인프라 필요
- 결제/구독 시스템 -- 이유: 사내 도구 용도이므로 과금 불필요
- 실시간 알림 -- 이유: MVP에서는 페이지 새로고침으로 충분
- 모바일 앱 -- 이유: 반응형 웹으로 대응. 네이티브 앱은 수요 확인 후 결정
